# rattles

Minimal, no-std terminal spinners for Rust, built on ratatui.

- **Repo:** [vyfor/rattles](https://github.com/vyfor/rattles)
- **Stars:** ~870 (2026-04-16)
- **Created:** 2026-04-10
- **Language:** Rust
- **Topics:** animation, cli, no-std, ratatui, rust, spinner, spinners, terminal, tui

## Why It Matters
- Tiny, focused: spinners only, but well-built. The kind of dependency you actually want.
- `no-std` support means it slots into embedded and constrained environments where most TUI libs won't.
- Quietly useful for the [[claw-code]] / TUI-tooling cohort — every CLI agent eventually needs a spinner.

## Key Takeaways
- ratatui ecosystem citizen — no reinvention.
- `no-std` = unusual reach for a TUI helper.
- A representative example of the "small, sharp Rust crate" pattern in trending data.
