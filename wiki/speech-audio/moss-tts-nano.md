# MOSS-TTS-Nano

Open-source multilingual tiny speech generation model. **0.1B parameters**, optimized for **CPU execution and real-time streaming** — no GPU required. Supports voice cloning, Chinese + English, multi-modality.

- **Repo:** [OpenMOSS/MOSS-TTS-Nano](https://github.com/OpenMOSS/MOSS-TTS-Nano)
- **Stars:** ~1.1K (2026-04-16)
- **Created:** 2026-04-10
- **Language:** Python
- **Topics:** audio-tokenizer, chinese, english, multi-modality, multilingual, realtime, streaming-audio, tts, voice-clone

## Why It Matters
- 0.1B-param TTS that runs in real time on CPU is a meaningful accessibility step — TTS without GPU dependence opens up local/edge deployments.
- Voice cloning + multilingual coverage at this size is unusual; most comparable open models are 1B+.
- Streaming output makes it usable for interactive agents, not just batch synthesis.

## Key Takeaways
- 0.1B params, CPU-real-time, voice cloning — the headline trio.
- From OpenMOSS, an established Chinese open-source NLP group.
- Not directly an agent tool, but enables agents that need a voice without a GPU bill.
