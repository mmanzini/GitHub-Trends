# html-ppt-skill

"HTML PPT Studio" — a Claude Code AgentSkill for building professional HTML presentations. Ships with 24 themes, 31 layouts, and 20+ animations.

- **Repo:** [lewislulu/html-ppt-skill](https://github.com/lewislulu/html-ppt-skill)
- **Stars:** ~1.5K (2026-04-20)
- **Created:** 2026-04-15
- **Language:** HTML

## Why It Matters
- Demonstrates the maturity bar for skills: a useful skill ships **assets** (themes, layouts, animations), not just prompts.
- HTML-as-presentation-format keeps output portable, web-native, and live-editable — different trade-offs than PowerPoint or Google Slides.
- Part of the broader [[design-systems-overview|design-for-AI]] trend: expressive output formats curated for agent consumption.

## Key Takeaways
- 24 themes × 31 layouts × 20+ animations — substantial library, not a stub.
- HTML output composes well with web tooling (deploys, embeds, sharing).
- A useful blueprint for "expressive media" skills: bring your own asset library.
