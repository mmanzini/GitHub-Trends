# video-use

Edit videos by conversing with [[claude-code-overview|Claude Code]]. Removes filler words, applies color grading, adds subtitles, and generates animation overlays via LLM-driven agentic editing. Built by the browser-use team.

- **Repo:** [browser-use/video-use](https://github.com/browser-use/video-use)
- **Stars:** ~1.1K (2026-04-18)
- **Created:** 2026-04-12
- **Language:** Python

## Why It Matters
- Extends the "talk to your tools" pattern from web (browser-use) to video editing.
- Demonstrates that **media editing** is a viable agent domain — not just code or research.
- Same authors as [[browser-harness]] — they're building a portfolio of single-domain agents with sharp scopes.

## Key Takeaways
- Conversational interface to non-trivial editing operations.
- Combines deterministic video pipelines with LLM-driven decisions (when to cut, what overlay to add).
- A useful template for "agentic editor" UX in any media domain.
