# GitHub Trending – 2026-04-26 (Sunday)

## Editor's Take
Three of this week's top ten repos are explicitly Claude Code Skills — `guizang-ppt-skill`, `garden-skills`, and `Kami` — signalling that the Skills format is fast becoming a first-class publishing layer on top of Claude, much like how npm packages or VS Code extensions became an ecosystem unto themselves. For an AI/agent content creator, this is the story: the toolchain around Claude is now self-replicating, with developers shipping opinionated, single-purpose capabilities that slot directly into the agent's context. A second pick worth attention is `Einsia/OpenChronicle`, a local-first, MCP-compatible agent memory system that captures real macOS context (open apps, active work, decisions) and makes it queryable by any tool-capable LLM — it fills a gap that closed-source memory products own today and could be a strong standalone video topic.
