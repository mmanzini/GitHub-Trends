# codeburn

Interactive TUI dashboard for AI coding cost observability. Tracks token usage and spend across Claude Code, Codex, and Cursor sessions in real time.

- **Repo:** [getagentseal/codeburn](https://github.com/getagentseal/codeburn) (originally [AgentSeal/codeburn](https://github.com/AgentSeal/codeburn))
- **Stars:** ~2.7K (2026-04-19) — sustained #1–#3 weekly trend over 2026-04-15 → 2026-04-19
- **Created:** 2026-04-13
- **Language:** TypeScript
- **Topics:** ai-coding, claude-code, cli, codex, cost-tracking, cursor-ide, developer-tools, observability, terminal-ui, token-usage

## Why It Matters
- Fills a gap **everyone** with a Claude Code / Codex / Cursor habit feels: "where did all my tokens go?"
- TUI keeps it fast and terminal-native, matching the workflow it instruments.
- Multi-vendor support (Claude + Codex + Cursor) is the norm now — single-vendor cost tools are already obsolete.

## Key Takeaways
- Real-time cost surface = behaviour change. Once you can see spend per task, you optimize for it.
- Pairs naturally with [[caveman]] (token-reduction skill) — measure savings empirically.
- Strong content angle: cost transparency turns abstract "AI bills" into concrete numbers viewers can react to.
