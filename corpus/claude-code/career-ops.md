# career-ops

AI-powered job search system built on [[claude-code-overview|Claude Code]]. Bundles 14 skill modes, a Go-based dashboard, PDF generation, and batch processing for resumes and interview prep.

- **Repo:** [santifer/career-ops](https://github.com/santifer/career-ops)
- **Stars:** ~36K (2026-04-20)
- **Created:** 2026-04-04
- **Language:** JavaScript (Go for dashboard)

## Why It Matters
- One of the highest-profile examples of a **vertical agent product** built entirely on Claude Code skills.
- Demonstrates the multi-skill packaging model: 14 specialized modes (resume tailoring, interview prep, etc.) coordinated under one agent UX.
- Validates that Claude Code can be the runtime for end-user applications, not just a dev tool.

## Key Takeaways
- 14 skill modes — granular task decomposition is a viable architecture.
- Go dashboard + PDF generation = polish layer on top of the agent core.
- Batch processing — agents can run pipelines, not just one-shot prompts.
